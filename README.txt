버킷 실행자 사용법
1. bukkit.bat 파일을 실행합니다.

사용자 사용법
1-1. 버킷 위치에 start.txt 파일이 추가되면 서버가 시작되며 시작될 시 start.txt는 삭제처리 됩니다. (서버 종료시 자동시작 안됨)
1-2. 버킷 위치에 restart.txt 파일이 추가되면 서버를 stop 하여도 다시 시작합니다. (서버 종료시 자동시작)

파일 소개
"start.txt" 및 "restart.txt"파일만을 추가만으로 서버를 실행할 수 있도록 만든 마인크래프트 버킷(BAT)입니다.
버킷 실행자(섭돌이, 등)은 컴퓨터가 켜져만 있으면 다른 이용자가 FTP, 등 파일 전송으로 서버를 킬 수 있도록 간편화 된 시스템입니다.
버킷 실행자는이 bukkit.bat을 바로가기를 만들어서 컴퓨터 부팅시 실행되게 하면 더 편리하게 사용할 수 있습니다.
